A open-source music player with Electron

### Prerequisites

* npm

### Installing

```
git clone https://github.com/pedr0ni/open-player.git
cd open-player
npm install
```

## Running the tests

```
npm start
```

## Deployment

```
electron-packager open-player --platform=win32 --arch=x64
```

## Built With
